Frequently Asked Questions

Is my data secure?

Yes, your data is secure. All data is stored locally on your server. We only use the Google Gemini API for query generation, and no data is sent to external servers except for the query text.

What file formats are supported?

We support CSV and Excel files (.xls, .xlsx). The system automatically detects the structure and creates appropriate database tables.

Can I upload multiple datasets?

Yes! You can upload multiple datasets and manage them simultaneously. Simply use the Upload Dataset button in the sidebar to add more databases.

What if my query is complex?

Our AI is powered by Google Gemini, which can handle complex queries. You can ask for aggregations, filtering, grouping, joins, and more - all in natural language.

Is there a cost?

NLQ Assistant is open source and free to use. You only pay for Google Gemini API usage, which is very affordable for most use cases.
